php计算字符串长度

                    $str1 = 'afdf.3424?gdfd;';//ASCII编码
                    $str2 = '我是a3.谁说的分。';//UTF-8
                    var_dump(strlen($str1));    //15
                    var_dump(strlen($str2));    //24=3 x 7 + 3
                    var_dump(mb_strlen($str1)); //15(根据当前字符的编码自动计算)
                    var_dump(mb_strlen($str2)); //10(查看当前字符的编码mb_detect_encoding($str))
                    var_dump(mb_strlen($str2, 'UTF-8')); //10
前提：安装PHP的时候安装了mbstring扩展

chunk_split:

                    $str = '我们tomorrow';
                    var_dump(chunk_split($str, 4, '?'));//结果：我�?��to?morr?ow?。每隔3个字符插入一个符号（因为一个汉字长度占3，[们]字被强行切割了）

                
explode:

                    $str = 'one|two|three|four|five';
                    var_dump(explode('|', $str));//结果：["one","two","three","four","five"]，全部分割
                    var_dump(explode('|', $str, 3));//结果：["one","two","three|four|five"]，部分分割。只分割成3份。
                    var_dump(explode('|', $str, -2));//结果：["one","two","three"]，只要部分分割结果。不要最后2个
